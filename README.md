# Conversor de Temperatura

Programa em Python que converte valores de temperatura entre **Celsius**, **Fahrenheit** e **Kelvin**, direto pelo terminal.

## Funcionalidades

- Conversão de Celsius para Fahrenheit e Kelvin
- Conversão de Fahrenheit para Celsius e Kelvin
- Conversão de Kelvin para Celsius e Fahrenheit
- Validação de entradas inválidas (o programa pede novamente caso o usuário digite algo que não seja um número)
- Menu interativo em loop, até o usuário escolher sair

## Tecnologias

- Python 3

## Como rodar

```bash
git clone https://github.com/Lucassm04/Conversor-de-Temperatura.git
cd Conversor-de-Temperatura
python conversor_de_temperatura.py
```

## Exemplo de uso

```
=== Conversor de Temperatura ===
1 - Celsius para Fahrenheit e Kelvin
2 - Fahrenheit para Celsius e Kelvin
3 - Kelvin para Fahrenheit e Celsius
0 - Sair
Escolha uma opção: 1
Digite a temperatura em Celsius: 25
25.0ºC = 77.00ºF
25.0ºC = 298.15K
```
